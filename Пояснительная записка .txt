Генератор и проверщик паролей 

Программа для проверки пароля пользователя и генерации пароля по запросу пользователя. 

 

Примененные технологии 

В программе присутствует три кнопки по нажатию на которые совершаются разнообразные операции.  

    1. При нажатии кнопки “Сгенерировать пароль”, появляется диалоговое окно типа QInputDialog, куда пользователь может ввести желаемую длину пароля, по умолчанию длина 11(если длина будет меньше 8, будет возбуждено исключение MyCustomError, и пользователю надо будет ввести длину повторно). Далее в поле вывода типа QTextEdit выводится сгенерированный пароль. 

    2. При нажатии кнопки “Проверить пароль” пользователь может ввести пароль в поле типа QLineEdit свой пароль, далее в поле типа QPlainTextEdit будет вынесен вердикт проверки пароля. Сложность пароля оценивается по нескольким пунктам: длина, наличие строчных и не строчных букв, наличие цифр и специальных символов, а также проверка на наличие в базе данных с небезопасными паролями. 

    3. Также у пользователя имеется возможность загрузить файл с несколькими паролями для проверки сразу нескольких паролей. Это можно сделать по нажатию кнопки “Загрузить файл с паролями“. По нажатию этой кнопки будет открываться диалоговое окно типа QFileDialog, где пользователь сможет выбрать файл формата .txt и на поле вывода будет вывед вердит по проверке каждого пароля. В случае загрузки пустого файла будет возбуждено исключение MyCustomError2 и выведено соответствующее сообщение. 

 

Установка и использование 

Достаточно просто скачать .exe файл с программой и запустить ее. Далее чтобы сгенерировать пароль необходимо нажать кнопку "Сгенерировать пароль" и ввести желаемую длину.
Чтобы проверить пароль нужно ввести пароль в окно для ввода и нажать кнопку "Проверить пароль". Если нужно проверить несколько паролей сразу, то это можно сделать с помощью загрузки файла с паролями. Для жтого необходимо нажать кнопку и загрузить файл с паролями с помощью появившегося диалогового окна. Далее будет выведен вердикт в поле вывода (перед этим оно очистится).
